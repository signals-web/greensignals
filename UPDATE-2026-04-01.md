# GreenSignals Update — April 1, 2026

## Nearby Signs

- **Hover highlighting** — hovering a sign in the Nearby panel highlights its marker on the map so you can quickly locate it
- **Persistent panel** — the Nearby panel is now a proper column alongside the sign card instead of an overlay. Clicking NEARBY shifts the layout to accommodate it; the sign card and map resize to fit. No more content getting covered
- **Smarter filtering** — limited to the 6 closest signs, prioritizing those with shared destinations, to reduce clutter in dense areas like Main Campus
- **Tighter zoom** — the sign card map stays focused on the current sign's destinations instead of zooming out to fit every nearby marker

## Building Name Standardization (new)

A full building name review and approval system, synced in real-time via Firebase.

### Building Names view

| Column | Description |
|---|---|
| **Building Name** | Click to edit the official display name |
| **Short Name** | Abbreviated version for space-constrained signs |
| **Original Name** | Read-only — the name as it appears in sign data |
| **Signs** | Number of sign destinations using this name |
| **Status** | Pending / Approved / Flagged / Deprecated |

- **Inline editing** — click any Building Name or Short Name cell to edit. Changes save on blur and sync to all reviewers instantly
- **Notes & aliases** — click the pencil icon to expand a notes row for aliases, style comments, and naming rationale
- **Status workflow** — approve, flag, deprecate, or reset each name. All statuses sync in real-time across reviewers
- **Search** — matches across official name, short name, original name, and notes/aliases
- **Status filter** — dropdown to view only pending, approved, flagged, or deprecated names

### 77 short names pre-generated for review

Short names follow these rules:
- Keep the core recognizable token people actually say out loud
- Drop generic facility words (Building, Center, Hall) unless they disambiguate
- Drop long honorific/donor fragments, admin qualifiers, redundant location qualifiers
- Use established acronyms where they exist

Examples:

| University Memorial Center | UMC |

All short names are editable in the Building Names view — review and adjust as needed.

### Sign card integration

- Destinations display the **official name** when one has been set (indicated by a gold left border; hover to see the original)
- **Nudge signs** automatically use the **short name** since they have less physical space
- **Deprecated names** appear with strikethrough styling

### Autocomplete (destination editing)

- Deprecated building names show a red "deprecated" warning in the dropdown
- Typing an alias (from the notes field) surfaces the official name as a suggestion

## UI Polish

- **Octicon icons** — Building Names action buttons now use GitHub Octicon SVGs (check, flag, X, history, pencil) at 28×28px
- **Light mode contrast** — back button darkened for visibility against the cream background
- **Responsive layout** — sidebar, right panel, and nearby column shrink proportionally at 1400px, 1200px, and 1024px breakpoints
